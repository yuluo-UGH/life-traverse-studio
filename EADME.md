# 给试用朋友的一封小说明  
——关于「遍历人生 · 老年工作室 v0.7（本地版）」

谢谢你愿意帮忙试用这个小程序。

它不是游戏，也不是打卡软件，而是给大约 **50–80 岁**的人准备的一间小小“工作室”，让你把这一段人生里 **想留下的东西**，慢慢写清楚、放好。

我们做的事很简单：

> 给你一个固定的地方写，  
> 帮你分门别类整理，  
> 将来你自己、家人、朋友再翻出来时，看得懂、有条理。

它只做 **记录和整理**，**不会** 做医疗、理财、法律等决策。

---

## 一、四个「工作室」各管什么？

### 1. 生活 & 回忆工作室  
（精彩生活 / 温馨回忆）

这里既记 **当下的生活**，也慢慢长成一份 **“个人回忆录”**。

- 一顿饭、一趟路、一句在意的话，都可以写进来；  
- 每次写一小段就行，后来再回头补时间、地点、人物；  
- 久而久之，这些卡片就会变成一份 **给自己和家人看的回忆录**：  
  不是为了出版，而是让“那时候我是怎样的人、遇到什么事”，有一个清楚版本。

### 2. 艺术工作室  
（作品 / 旋律 / 小说骨架）

给你放 **作品和灵感** 的地方：

- 写过的诗、小文章、画的构思、拍照心得；  
- 可以用「记录一段旋律」先把哼出来的调子记成简单数字；  
- 也可以用「小说骨架（三问）」先搭一个人物和故事框架，后面慢慢填。

不要求专业，只是让这些东西 **别在脑子里一闪而过就消失**。

### 3. 思想工作室  
（人生观 / 教育观 / 信仰体会）

写清楚：“**我是怎么想的**”。

- 对孩子、对钱、对工作、对某种信仰、对某段经历；  
- 不求标准答案，只是帮你把那一套想法理出来，将来自己或后辈再看时能看懂。

这里只做整理，不帮任何阵营“站队”。

### 4. 技术 & 创意工作室  
（项目经验 / 技术备忘录 / 点子草图）

不只是给“老专家”准备的，也欢迎 **有创意、有想法的人**：

- 你这一生做过的项目、积累的经验；  
- 你想到过的设计、方案、点子草图，还没来得及做；  
- 都可以拆成一张张卡片：写背景、写做法、写哪里难、写你觉得更好的改法。

既是留给后人，也是在帮自己，把一辈子想过、做过的事再看一遍。

## 二、v0.7 相比上一版，有什么变化？

这次 v0.7 还是 **纯本地版**，没有云端账号、也没有远程 AI，只在这一台设备上工作。  
主要多了三件事：

1. **一个“简单入口”**  
   - 打开工作室后，可以先只看到三个大按钮：  
     - 「说一说」：打开一个大输入框，  
       你可以点键盘里的语音按钮（🎤），直接说出来，自动变成文字；  
     - 「拍一张」：以后会支持照片卡（当前本地版先以内测为主）；  
     - 「写一小段」：只写几行文字就行，后面在工作室里可以慢慢补。  
   - 不用一开始就想：这算生活、艺术，还是技术。

2. **给卡片打上简单的“标签”**（可选）  
   - 比如：和谁有关？`自己 / 爱人 / 子女 / 孙子孙女 / 老同学 / 同事 …`  
   - 大概是什么类的事？`日常生活 / 工作项目 / 作品创作 / 信仰社团 / 身体心情`  
   - 以后可以按标签来找：比如“点一下孙子孙女”，相关的卡片都会出来。

3. **多了一种“导出备份”的办法**  
   - 当前版本的数据只保存在 **这一台手机 / 这一台电脑的浏览器本地**；  
   - 如果手机坏了 / 丢了 / 重装浏览器，数据有可能丢失；  
   - 现在可以请家人帮你：  
     - 点一下「导出备份」按钮；  
     - 把导出的备份文件存到家人的电脑、U 盘或云盘里；  
     - 换手机时，再用「导入备份」恢复回来。

## 三、怎么输入内容？（不会打字也可以）

### 1. 语音输入（推荐）

只要你的手机键盘里有“语音输入”（豆包输入法、讯飞、系统自带都可以）：

1. 点「说一说」，进入大输入框；  
2. 点 **键盘上的语音按钮（🎤）**；  
3. 对着手机说：  
   - “今天和孙子在小区玩了一圈……”  
4. 说完之后停一下，确认文字已经出现，再点保存。

> 工作室本身并不做语音识别，  
> 只是接住「你的输入法已经转好的文字」，  
> 然后帮你存成一张卡片。

### 2. 文字输入（少量为主）

- 适合本来就习惯打字的朋友；  
- 或者由子女在旁边帮你敲字。

## 四、关于保存和备份（非常重要）

### 1. 当前版本的底线

- 所有内容都存放在 **这一台设备的浏览器本地**；  
- 不会自动上传到服务器，也不会同步到别的手机上；  
- 换手机 / 恢复出厂设置 / 清理浏览器数据，有可能导致内容丢失。

### 2. 建议的备份做法（需要家人帮忙）

在工作室里有一个「导出备份」按钮：

1. 请家人帮你点击「导出备份」；  
2. 程序会生成一个类似 `elder_studio_backup_日期.json` 的文件；  
3. 把这个文件保存到：  
   - 家人的电脑、U 盘，  
   - 或微信/网盘聊天里，  
   作为一份“家庭备份”。

以后如果手机坏了/换了：

1. 在新手机上再次打开老年工作室；  
2. 让家人点「导入备份」，选中之前保存的那个备份文件；  
3. 之前的卡片就能恢复回来。

## 五、我们刻意坚持的几件事

1. **不哄人、也不喊口号**  
   - 不会说太多“您太伟大了”这种空话；  
   - 更多会说：“那我们先把这一件事写清楚，好不好？”

2. **一次只做一点，但要做得实在**  
   - 不要求你天天打卡、一次写很多；  
   - 但希望每次都留下一个**看得见的小成果**：  
     一张生活卡、一条旋律、一个故事大纲、一份技术/创意笔记。

3. **边界清楚：不做医疗、理财、法律答复**  
   - 身体、用药、手术 → 请一定找医生；  
   - 投资、借钱、买理财 → 请和家人、专业机构商量；  
   - 法律纠纷 → 请找律师。  
   工作室能做的，是帮你 **把问题、经历、想法写清楚**，  
   让你和家人、专业人士沟通时，心里更有数。

## 六、你可以怎么试用？

可以随便选一个开始：

- **进「生活 & 回忆工作室」**：  
  写一件最近最在意的小事，给它建一张卡片。

- **进「艺术工作室」**：  
  试一次「记录一段旋律」，或者用「小说骨架」写一个你心里忘不掉的人物。

- **进「技术 & 创意工作室」**：  
  写一件你这一生最想留下的“做法”或“点子”：  
  为什么重要？当时怎么做的？现在会怎么改？

- **用“简单入口”随手记：**  
  想到什么就点「说一说」或「写一小段」，先留下来，以后再整理。

不用写得好，愿意开始，就已经很珍贵。

## 七、我们最想听到你的反馈

如果你愿意帮忙说说感受，我们特别在意这些问题：

- 你能不能 **自己顺利进来、再回来**？  
- 字体和按钮，看起来 **累不累眼**？  
- 用语音输入 / 写一小段，保存卡片，再打开看，**这条路好不好懂**？  
- 按照“人”（比如孙子孙女）或“主题”（比如工作项目）去找卡片，**好不好用**？  
- 整体感觉，是更像一个 **认真、值得信任的帮手**，  
  还是像一个在催你“交作业”的软件？

你可以直接说哪里别扭、哪里舒服、哪里完全看不懂，我们都会认真记下。

如果有一天，这个老年工作室真能帮一部分人，把自己的生活、回忆、想法和创意留下来一点点，那里面肯定有你这次试用的一份功劳。
